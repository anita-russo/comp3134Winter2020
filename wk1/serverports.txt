1) What is the definition of server port?
Answer: A communication endpoint that a server uses to make its services
available to the internet using numbered ports (one for each service)

2) What is server port range?
Answer: 0 to 65535

3) How many bits does number above represent?
Answer: 16

SMTP (30)
